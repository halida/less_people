# 少生孩子影响分析

宏观：
财富由人创造
不能积累的财富：物资，服务
可以积累的财富：基建，科学技术，知识

模拟
产能 = 人口 * 劳动生产率
未来人口减半，劳动生产率x1.5
未来产能 = 0.5 * 1.5 现在产能 = 0.75 现在产能

假设货币不变，通货膨胀 33%

劳动者收入提升，体力劳动者也一样

不生小孩，积蓄钱，损失1/3
生小孩，无反馈

模拟4/2/1

人口减少一半
30元吃饭，服务员3000收入
一半餐馆没服务员关门，每个餐馆挤爆，收入翻倍
餐馆加薪到5000请服务员，菜涨价到50元

所有人的收入都会高起来，消费也高起来
钱不值钱，
